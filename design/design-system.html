Personal Finance · v1.0 · claro & escuro

Controle o seu dinheiro com clareza.

Sistema de tokens e componentes para um app de finanças pessoais. Azul-escuro para saldo, estrutura e confiança; laranja-escuro para as ações que movem o dinheiro. Verde para o que entra, vermelho para o que sai. Compatível com shadcn/ui, nos temas claro e escuro.

01 — Fundamentos

Marca

Um gráfico ascendente como símbolo — crescimento, patrimônio, dinheiro que sobe. Use o lockup completo por padrão; o ícone com fundo em espaços quadrados (avatar, favicon); o símbolo sozinho quando já houver contexto da marca.

04 — Movimento

Movimento & animações

Animação com propósito, no padrão do mercado (motion.dev / Framer Motion). Movimento explica de onde algo veio, confirma uma ação e guia a atenção — nunca decora à toa. Curto e discreto: se você repara na animação, ela está longa demais. Sempre respeite prefers-reduced-motion.

Duração
Instantâneo100ms · press, toggle
Rápido160ms · hover, chips
Base220ms · overlays, cards
Lento320ms · drawer, sheets
Entrada600–900ms · tween de valor
Easing
Standardcubic-bezier(.2,0,0,1)
Transições gerais — o default de tudo.
Decelerate (out)cubic-bezier(0,0,.2,1)
Entrada de elementos — chega rápido e assenta.
Accelerate (in)cubic-bezier(.4,0,1,1)
Saída de elementos — sai da tela ganhando velocidade.
Springstiffness 380 · damping 34
Drawer, arrastar, gestos — no Motion, type: 'spring'.

Com Motion (Framer Motion)
Overlay — entrar & sair
<motion.div
  initial={{ opacity: 0, scale: .96 }}
  animate={{ opacity: 1, scale: 1 }}
  exit={{ opacity: 0, scale: .96 }}
  transition={{ duration: .18, ease: [.2, 0, 0, 1] }} />
Drawer — spring pela lateral
<motion.aside
  initial={{ x: '100%' }} animate={{ x: 0 }}
  transition={{ type: 'spring', stiffness: 380, damping: 34 }} />
Lista — stagger dos filhos
<motion.ul
  variants={{ show: { transition: { staggerChildren: .06 } } }}
  initial="hide" animate="show" />
Acessibilidade
Sempre respeite prefers-reduced-motion: reduza transições a 0ms ou anime só opacidade — nunca movimento/scale.
@media (prefers-reduced-motion) {
  * { animation: none !important;
      transition: none !important; }
}
No Motion, use o hook useReducedMotion() e desligue x / scale, mantendo o fade.
